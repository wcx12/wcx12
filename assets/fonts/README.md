# Self-hosted fonts

The site serves Latin subsets of these open-source typefaces locally:

- Space Grotesk, Google Fonts v22, licensed under SIL Open Font License 1.1.
- JetBrains Mono, Google Fonts v24, licensed under SIL Open Font License 1.1.

The accompanying `OFL-*.txt` files contain the license text. The WOFF2 files
were retrieved from the URLs published by the Google Fonts CSS2 API and are
used without modification.
